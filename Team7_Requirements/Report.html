Team 7 - Project 1 - Chocoholics Anonymous

For the first project of this course, we were tasked with creating individual html files containing information about our team and a single report file that contained a use case diagram for ChocAn, a term glossary, and descriptions for each of our use cases. For one of our use cases, called “Pay fee,” an assumption we made was that the ChocAn company might have an online presence that could be used to pay the monthly fee. We compensated for this assumption by adding an extension point in the description where this path could be taken if possible. We each chose a specific part of the report to provide input toward alongside working on our own individual reports. Each team member was given three tasks to complete. The report compiler and submitter for Team 7 was Warren Flenniken. From this project, we gained a greater understanding of collaborating through Bitbucket. We also became more familiar with using UMLetino to create larger scale use case diagrams.
